What is MeshCore?

Off-grid messaging that needs no cell towers, Wi-Fi, or internet.

MeshCore is an open-source firmware and app ecosystem for long-range, low-power communication over LoRa. Small, inexpensive radios relay text messages for one another, forming a self-healing mesh that keeps working when conventional infrastructure is unavailable — useful for emergencies, events, hiking, and community resilience.

Unlike a simple flood network, MeshCore uses lightweight managed routing: nodes learn paths through the mesh, so messages take efficient hops instead of being rebroadcast by everyone. Repeaters like this one are the backbone — placed high and powered around the clock, they carry traffic over distances a handheld node could never reach alone.

Join the mesh

From zero to on-air in four steps.

  1. Get a supported radio Any MeshCore-compatible LoRa board for the US band — e.g. Heltec V3, RAK WisBlock, LilyGO T-Deck, or a Seeed SenseCAP solar node.
  2. Flash MeshCore firmware Use the web flasher to install the Companion firmware on your node — no toolchain required.
  3. Install a client Pair your node with the MeshCore app (Android, iOS, or web client) over Bluetooth, USB, or Wi-Fi. Download links are at meshcore.io.
  4. Set the radio preset Choose the USA / Canada preset with the parameters shown above. Once you're in range, the nearest Burlington Mesh repeater will appear in your contact list and relay your messages.

MeshCore node roles

The building blocks of a MeshCore network.

Companion

Your personal node

A radio paired to your phone or computer. You read and send messages through the client app; the node handles the radio.

Repeater

Range extender

An always-on node — like this one — that forwards traffic between others, stretching the mesh across a town or region.

Room Server

Group chat & bulletins

Hosts persistent channels and stores messages, so members can catch up on conversations even after being offline.

Sensor

Telemetry

Reports readings such as temperature, voltage, or GPS over the mesh — handy for monitoring remote, off-grid equipment.
